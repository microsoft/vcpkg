# vcpkg_clean_msbuild

The latest version of this document lives in the [vcpkg repo](https://github.com/Microsoft/vcpkg/blob/master/maintainers/vcpkg_clean_msbuild.md).

Clean intermediate files generated by `vcpkg_install_msbuild()`.

## Usage
```cmake
vcpkg_clean_msbuild()
```
